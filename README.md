# listing-mini-project-frontend

## Frontend setup

To run the frontend: 
  npm run dev

To start the frontend with MSW (must be run alongside npm run test):
  npm run dev-msw

To start the frontend test:
  npm run test

To create the mockServiceWorker.js file
  npm run msw-init



## Backend setup

To run the backend:
npm run start
